Homepage

For the Fahrrad-XXL homepage redesign, I prioritized a user-friendly interface with a clean, contemporary look that reflects the brand's image. The layout showcases a responsive hero section, prominent promotional content, and direct call-to-action elements to facilitate user engagement and conversion. Real-time inventory levels enhance the shopping experience by providing immediate product availability. Trust signals, such as customer reviews and security badges, are strategically positioned to establish credibility.

HR Portal

The HR portal was designed as an intuitive gateway for prospective applicants, emphasizing Fahrrad-XXL's commitment to a straightforward recruitment experience. It features a dedicated 'XXL Jobs' menu, ensuring job seekers have instant access to the opportunities available. The user journey from discovering job openings to submitting an application is fluid, supported by a robust backend system that efficiently handles data collection and candidate notifications.

Job listing Page

The job listing page is a focal point within the portal, showcasing all available positions. Jobs are neatly categorized by department with a visual count of openings, facilitating a user-friendly browsing experience. Each listing is interactive, directing users to detailed job descriptions that provide a comprehensive view of the role and its requirements.

Store Location Page

A store location page design concept has an interactive map mirroring the branch site maps, allowing applicants to search for jobs by geographical preference. The page not only displays the number of positions available at each location but also provides a direct link to the store's page for detailed insights, enhancing local employer branding.

Application page

The application page is designed for efficiency, with a form that connects to both the company's database and professional networks like Xing and LinkedIn. This integration allows for a seamless application process, with the system capturing all necessary details and supporting document uploads. The form is intuitive, requesting information in a structured manner to ensure a smooth applicant experience.
